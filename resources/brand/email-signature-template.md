## Setup Instructions

### Gmail
1. Go to Settings → See all settings
2. Scroll to "Signature" section
3. Create new signature named "Claw Forge Standard"
4. Paste HTML or plain text version
5. Set as default for new emails
6. Save changes

### Apple Mail
1. Mail → Preferences → Signatures
2. Click + to create new
3. Paste signature text
4. Set default account
5. Close preferences

### Outlook
1. File → Options → Mail
2. Click "Signatures"
3. New → Name it "Claw Forge"
4. Paste content
5. Set as default
6. OK

### Mobile (iPhone/iPad)
1. Settings → Mail → Signature
2. Select "Per Account"
3. Paste short signature version
4. Keep it brief for mobile

## Usage Guidelines

### Always Include
- Full name
- Company name
- Website link
- Email address

### Optional Based on Context
- AI Avatar Studio link (product emails)
- Phone number (if you want calls)
- Social media (if relevant)

### Never Include
- Personal social media (keep it business)
- Unnecessary quotes
- Too many links (max 3)
- Images (can break in some clients)
